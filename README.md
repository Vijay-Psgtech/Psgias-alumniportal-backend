# PSG IAS Alumni Portal – Backend

This repository contains the **Express/MongoDB backend** for the PSG IAS Alumni Portal application.  
It exposes RESTful APIs for authentication, alumni data, donations and admin dashboards.

## 📁 Project Structure

\`\`\`
config/
controllers/
middleware/
models/
routes/
Server.js
package.json
\`\`\`

## 🚀 Getting Started

### Prerequisites

- Node.js v16+  
- MongoDB (local or Atlas)  
- npm (comes with Node)

## 🛠 Core Features

- **Admin authentication & management**  
- **Alumni CRUD operations**  
- **Protected routes with JWT**

## 📦 Models

- **Admin** – admin users  
- **Alumni** – alumni records
